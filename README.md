# 🩸 BloodNet – Online Blood Donation System

## 📌 Project Description
BloodNet is a web-based platform that helps users find blood donors and request blood easily. It connects donors and recipients in a fast, simple, and user-friendly way.

---

## 🎯 Objectives
- Provide quick access to blood donors
- Allow users to request blood by type
- Encourage voluntary blood donation
- Spread awareness about saving lives

---

## ⚙️ Features
- 🔍 Search blood by type (A+, B-, O+, etc.)
- 🧑 Donor listing system
- 📋 Blood request form
- ❤️ Blood donation form
- 📊 Interactive dashboard (UI-based)
- 🎨 Modern graphical interface

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Advanced UI & animations)
- JavaScript

---

## 💻 How to Run the Project
1. Download or clone this repository
2. Open the project folder
3. Double-click on `index.html`
4. Website will open in your browser

---

## 🌐 Deployment
This project can be hosted using GitHub Pages.
